What is callback function?
A callback function is a function passed into another function as an argument.You will callfunction inside and other function to complete the actionWhy we need callback function?Basically ,js is an event driven langauge.Instead of waiting for a response,javascript keep on executing the event while listen to the other event or process after thatcallback are way to make sure that certain code doesn't execute until other code get finished its execution.
